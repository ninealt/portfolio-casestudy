## Anotaciones de Diseño

### Principios Mantenidos

1. **Dark mode consistente** — Todo el sitio en tema oscuro
2. **Neón sutil** — Acentos que no saturan
3. **Respiración** — Espacio blanco (negro) generoso
4. **Jerarquía clara** — Tamaño, peso y color guían la atención
5. **Consistencia** — Componentes reutilizados, patrones repetidos

### Diferencias Desktop/Mobile

| Aspecto     | Desktop          | Mobile          |
| ----------- | ---------------- | --------------- |
| Navegación  | Sidebar + Header | Bottom tabs     |
| Skills      | Grid 4 cols      | Accordion list  |
| Timeline    | Completa         | Últimos 3 roles |
| Hero        | 2 cols           | 1 col stacked   |
| Animaciones | Full             | Reducidas       |
