# Sección 1: Introducción y Problema

> **Módulo 7 – Desarrollo de Portafolio de un Producto Digital (18h)**

## Objetivo del Módulo

Presentar un producto digital completo en un portafolio personal, utilizando herramientas tecnológicas y buenas prácticas de la industria, que permita potenciar el perfil profesional y demostrar las competencias adquiridas durante el curso.

---

## 🎮 El Concepto: Quest Log

Este proyecto nace de una necesidad personal: transformar un CV tradicional en algo que realmente **conecte** con quien lo ve.

### De lo Standard a lo Sazonado

Tenía un CV anterior que era correcto pero muy estándar. Cumplía su función, pero sentí que le faltaba **impulso**, le faltaba **sazón**. Estaba bien, pero no destacaba.

**¿La solución?** Integrar algo que me define como persona y profesional: **los videojuegos**.

### La Inspiración RPG

Pensé en crear una experiencia de rol, integrando conceptos sutiles de:

| Concepto RPG | Aplicación en el Portafolio |
|--------------|----------------------------|
| 🗡️ **Quests** | Proyectos y desafíos profesionales |
| 🧭 **Viajes** | Trayectoria profesional y aprendizajes |
| ⚡ **Habilidades** | Stack tecnológico y competencias |
| ⭐ **Skills** | Nivel de dominio en cada tecnología |
| 🎭 **Roles** | Perfiles híbridos (dev + diseñadora) |

> **Importante:** La inspiración es sutil, no saturación. No es un juego, es un portafolio profesional con personalidad.

---

## 🎨 Componentes Visuales

### Estética Neón + Línea Clásica

Los componentes elegidos combinan:

- **Luces contrastantes de neón** (cyan `#00d4ff` y gold `#ffd700`)
- **Línea sobria clásica** para mantener la profesionalidad
- **Dark mode** como base para reducir fatiga visual y destacar el contenido
