# Sección 6: Testeo y Mejoras

## Validación con Usuarios Reales

Esta sección documenta el proceso de testeo, feedback recibido y las iteraciones realizadas para mejorar el portafolio.

---

## Metodología de Testeo

### Enfoque

- **Usuarios de prueba:** 5 personas
- **Perfiles:** 3 reclutadores tech, 2 desarrolladores
- **Método:** Test de usabilidad remoto + encuesta
- **Duración:** 10-15 minutos por sesión

### Tareas de Test

1. Encontrar el stack tecnológico
2. Identificar años de experiencia
3. Encontrar información de contacto
4. Evaluar el diseño general
5. ¿Recomendarías/contactarías a este candidato?
