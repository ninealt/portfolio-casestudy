## Fundamentos Decisorios

### Basado en Investigación

| Insight | Decisión de Diseño |
|---------|-------------------|
| 62% revisa < 1 minuto | Hero con toda la info crítica |
| 78% busca stack primero | Sección de skills prominente |
| 92% rechaza errores | Testing riguroso obligatorio |
| Mayoría usa móvil | Mobile-first approach |
| Dark mode trending | Paleta oscura con acentos neón |

### Principios Guiadores

1. **Claridad sobre creatividad** — Lo creativo no debe obscurecer lo funcional
2. **Escaneabilidad primero** — La información debe ser digerible en segundos
3. **Mobile-first** — Diseñar para constraints, no solo adaptar
4. **Consistencia** — Sistema de diseño riguroso
5. **Autenticidad** — Que se sienta "mío" sin ser autocomplaciente
